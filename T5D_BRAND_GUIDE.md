# T5D Brand Guide

## Brand Identity

Project: Tec5uport Dimension (T5D)  
Classification: Omnichain Intelligent Support Layer  
Motto: One Platform. Infinite Support.  
Core tagline: The World's First Omnichain Intelligent Support Layer

## Symbol

The T5D symbol is a pentagon.

The pentagon represents structure, foundation, and the 5 in T5D.

## Five Pillars

Top: 5upport  
Upper Left: Community  
Upper Right: Intelligence  
Lower Left: Security  
Lower Right: Education

## Logo Direction

Remove separate pillar icons.

Use a dimensional pentagon with inside layers, subtle circuitry, and the T5D core mark.

The design should feel:

- Welcoming
- Trusted
- Intelligent
- Dimensional
- Structured
- Futuristic
- Not overly neon
- Not overly aggressive
- Not dull

Soft Cyan White: #9FD7FF  
Deep Ruby Red: #E21B3C


## Separate Website Pillar Icons

The core pentagon mark should remain clean with inside layers.

Separate pillar icons may be used across the website as interactive links, cards, navigation aids, or section markers.

Recommended icons:

- 5upport: headset/support signal
- Community: connected people/nodes
- Intelligence: neural core/network
- Security: shield/check
- Education: open book/learning path

These icons should use the same pentagon frame, royal blue, gunmetal silver, deep ruby red accent, and soft cyan glow.
